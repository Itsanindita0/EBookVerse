# **App Name**: EBookVerse

## Core Features:

- User Authentication: Secure user registration and login with Firebase Authentication, supporting email/password and Google Sign-In.
- E-book Upload and Management: Admin users can upload e-books with metadata (title, author, genre, description) and cover images.
- E-book Browsing and Search: Users can browse e-books, filter by genre/author, sort by title/rating, and search by keyword.
- Online Reader: Read books online with PDF.js, with controls for page navigation, zoom, bookmarks, text highlighting, and notes.
- Personal Library: Users can add e-books to their personal library, save reading progress, and add reviews/ratings.
- Recommendation Engine: AI-powered engine that tool recommends books to users based on their reading history, genre preferences, and ratings.
- User Profile and Statistics: Users can view their profile information, reading statistics (books read, pages read, time spent), and manage their library.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a calm, intellectual atmosphere.
- Background color: Light gray (#ECEFF1), subtly tinted towards the blue for consistency.
- Accent color: Vivid orange (#FF9800) to highlight important actions.
- Headline font: 'Playfair', serif, elegant headlines that contrast nicely with body text
- Body font: 'PT Sans', sans-serif, for clear and readable body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use modern, outline-style icons to represent different categories and actions.
- Emphasize a clean, spacious layout with clear separation of content sections.
- Subtle transitions and animations to provide a smooth and engaging user experience.
